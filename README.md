# Реализация макета для сайта заказов тортов в виде React-компонентов

В рамках данного домашнего задания мной был реализован макет одностраничного сайта для заказа кондитерских изделий, предварительно разработанный в среде Figma. Целью выполнения работы являлась отработка навыков вёрстки пользовательского интерфейса и переноса дизайнерского макета в код.

В процессе выполнения задания была выполнена декомпозиция визуальных частей веб-страницы на самостоятельные React-компоненты, что позволило структурировать проект и упростить повторное использование элементов интерфейса. Реализация компонентов осуществлялась на основе ранее написанного кода на языке разметки HTML с его последующей адаптацией под компонентный подход библиотеки React.

В результате выполненной работы был получен одностраничный веб-сайт, корректно отображающий структуру и дизайн макета, а также демонстрирующий практическое применение React-компонентов при разработке пользовательских интерфейсов.
